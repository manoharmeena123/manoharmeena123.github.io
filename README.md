## üöÄ Getting Started

### Prerequisites
- Modern web browser
- Local development server (optional)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/manoharmeena123/manoharmeena123.github.io.git
   ```

2. Navigate to the project directory:
   ```bash
   cd manoharmeena123.github.io
   ```

3. Open `index.html` in your browser or serve it locally:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   
   # Using PHP
   php -S localhost:8000
   ```
